Connect to Google Drive
Sign in with your Google account once. Your habit data syncs automatically to a file in your Google Drive — accessible on iPhone, iPad, Mac, anywhere.
1
Go to console.cloud.google.com and sign in with your Google account
2
Create a new project → enable Google Drive API → set up OAuth consent screen (External, your Gmail as contact)
3
Go to Credentials → Create OAuth 2.0 Client ID
Type: Web application
Authorised JS origins: your GitHub Pages URL
Authorised redirect URIs: your GitHub Pages URL
4
Copy your Client ID (ends in .apps.googleusercontent.com) and paste it below
Google OAuth Client ID
Your Client ID is stored locally in this browser only.
Data saves to habit-data.json in your Google Drive.
